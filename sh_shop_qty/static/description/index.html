Multiples of Quantity in Shop

Are you planning to define the multiples of quantities to the products which you are selling on your store build up with the website? Sometimes in the shop, some products come in a bunch of 6, 12, etc. You can set easily multiples of quantity No using this module. Suppose you want to set multiples of quantity is 5, it will default the value of the product in the shop. If customers put manual quantity (manually = 8)of the product so it will automatically take higher No of quantity(higher = 10). Customers can increase or decrease the quantity of products relative to default quantity(default is 5) in shop or cart. If the customer click button "Add to Cart" in the shop, product quantity automatically sets the default quantity. You can do multiples of quantities for multiple websites.
